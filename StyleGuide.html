Starter Template

To create a new page, start with this basic HTML structure. It includes the required Google Fonts and the CSS variables that define the site's look and feel.

template.html 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>New Page</title>

    <!-- Google Fonts -->
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
    <link href="https://fonts.googleapis.com/css2?family=Rajdhani:wght@600;700&family=Inter:wght@400;500;600&display=swap" rel="stylesheet">

    <!-- Core Styles -->
    <style>
        :root {
            --color-background: #0A0A0A;
            --color-text: #EAEAEA;
            --color-text-muted: #888888;
            --color-primary: #FF3B30;
            --color-surface: #141414;
            --color-border: #2A2A2A;
            --font-heading: 'Rajdhani', sans-serif;
            --font-body: 'Inter', sans-serif;
        }
        body {
            background-color: var(--color-background);
            color: var(--color-text);
            font-family: var(--font-body);
        }
        /* Add other base styles or component styles here */
    </style>
</head>
<body>
    <h1>Your Content Here</h1>
</body>
</html>
